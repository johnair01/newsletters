<objective>
Add the typed foundation for the learning surface: the fifth `learning` SurfaceTemplate (L1)
and the new typed `GlossaryBlock` whose term definitions are traced Claims, not strings (L2).
This plan ships ONLY the typed config + the new block model — no preset logic, no render, no
dogfood. It is the contract every later plan in this phase builds against.

Purpose: A fifth reader-facing surface is *config* (a SurfaceTemplate via the registry), and
the in-context glossary's faithfulness is *enforced by the type* — a term's definition is a
`Claim` (carrying `evidence: list[Trace]`), so an un-traceable definition literally cannot be
constructed. This locks the faithfulness crux into the schema before any logic exists.

Output: `templates.py` LEARNING preset (registered), `semantic.py` GlossaryTerm + GlossaryBlock
(in the Block union), and `tests/test_learning.py` scaffolding the registration + typing tests.
</objective>
