## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| typed model → rendered HTML | Claim/term/path text is interpolated into HTML. Untrusted-ish only in that curated content must still be escaped; every interpolation goes through _e() (existing pattern). Only path/slug-safe locator values enter an href (T-09-08); free text never does. |
| rendered page → reader's browser | The page must be self-contained: no auto-loading external URL (fonts are self-hosted relative woff2, Phase 11). A repo-blob content link the reader clicks is allowed; an auto-fetched off-box resource is not. |

| T-12-04b | Tampering / Info | A dead href="None" or off-box auto-loaded resource on the learning surface or path | mitigate | Reuse link_for_source (returns None → plain text, never a dead link) and the _fanout_row plain-text fallback for unresolved steps; test_learning_surface_and_path_make_no_external_call asserts no auto-loading external URL and only the theme-toggle script. |
